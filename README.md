## 🎮 โหมดการเล่น

### 📚 Practice Mode
- เลือกจำนวนข้อได้ตามต้องการ (10 / 25 / 50 / 100 / ทั้งหมด หรือกำหนดเอง)
- แสดง Feedback ✅/❌ ทันทีหลัง Submit
- มีปุ่ม **ข้าม** สำหรับข้อที่ไม่แน่ใจ
- **ไม่จำกัดเวลา**

### 🎯 Exam Mode
- จำลองข้อสอบ CEH จริง **125 ข้อ** สุ่มจากคลังรวม
- จับเวลา **4 ชั่วโมง** (240 นาที)
- เกณฑ์ผ่าน **70%** (88/125 ข้อ)

### 📖 Module Practice
- ทำข้อสอบแยกตาม **Module CEH v13** ทั้ง 20 บท

## 🌐 ฟีเจอร์แปลภาษาไทย

แอปใช้ **Anthropic Claude API** แปลโจทย์เป็นภาษาไทยแบบ On-demand

1. กดปุ่ม **🌐 แปล** ระหว่างทำข้อสอบ
2. ใส่ Anthropic API Key ครั้งแรก (รับได้ที่ [console.anthropic.com](https://console.anthropic.com/settings/keys))
3. ระบบจะแปลและ Cache ไว้ในหน่วยความจำ — กดซ้ำโดยไม่เรียก API อีก

> API Key จะหายเมื่อ Refresh หน้า — ไม่มีการบันทึกลงที่ใด
